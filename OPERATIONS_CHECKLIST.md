## 三、音频处理能力

### 1. 格式与兼容
- [x] 多格式支持 (WAV/MP3/M4A/FLAC)
- [x] Python 3.13 兼容 (soundfile + librosa 替代 pydub)
- [x] 采样率自动转换
- [x] 单/双声道自动适配

### 2. 母带处理
- [x] 语音引导: -16 LUFS 目标响度
- [x] 纯音乐: -14 LUFS 目标响度
- [x] 峰值限制: -1 dBTP
- [x] 等功率淡入淡出 (float32,避免内存翻倍)
- [x] 等功率交叉淡化 (float32)
- [x] 音乐避让 (ducking)
- [x] 削波保护

### 3. 安全防护 (本次审计新增)
- [x] 解码前采样率/声道/帧数预检 (≤192kHz / ≤8ch / ≤30min)
- [x] 最小裁剪保留 100ms / 最大循环拼接 200 次
- [x] 供应商响应字节上限 200MB / Content-Length 预检
- [x] MiniMax TTS WAV 帧数上限 5 分钟

## 五、基础设施

### 1. 服务架构

- [x] Python Worker (后台任务处理)

- [x] 管理员敏感操作二次邮箱验证
