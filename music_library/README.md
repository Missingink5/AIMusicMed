# 本地情绪音乐库

本目录包含主流程使用的 61 首正式 WAV 素材，共 7 类：

| 目录 | 情绪 | 数量 |
| --- | --- | ---: |
| `Anxiety/` | 焦虑 | 8 |
| `Happy/` | 喜悦 | 8 |
| `Hostility/` | 敌意 | 9 |
| `Love/` | 友爱 | 8 |
| `Pride/` | 自豪 | 9 |
| `Quiet/` | 平静 | 10 |
| `Sad/` | 忧郁 | 9 |

`local_music_library.py` 会扫描这些目录，根据用户输入映射情绪并选择整首音乐。主流程只使用本目录中的正式素材，不裁剪、不循环，也不回退到在线音乐生成模型或静音占位。

素材已按整曲综合响度统一：非焦虑类别约为 `-16 LUFS`，焦虑类别约为 `-23 LUFS`，保留约 7 dB 的产品设计差异；全部限制真峰值不高于约 `-2 dBTP`。后续新增或替换素材时应保持这一规则，避免再次对全库统一归一化而抹掉焦虑类别的音量差。

添加素材时请保持现有目录名称，并确认拥有演示、公开仓库及再分发所需的授权。不要将临时生成的 `music_*.wav` 或 `speech_*.wav` 放入本目录。

验证扫描结果：

```powershell
$env:PYTHONUTF8 = "1"
python local_music_library.py
```
