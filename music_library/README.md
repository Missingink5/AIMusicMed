# 音乐库目录结构

请将您的音乐文件按照以下情绪分类放入对应文件夹：

## 目录说明
```
music_library/
├── Hostility/     # 愤怒、敌对情绪的音乐 (20首左右，每首约1分钟)
├── Sad/           # 悲伤、低落情绪的音乐
├── Anxiety/       # 紧张、不安情绪的音乐  
├── Quiet/         # 平和、安静情绪的音乐
├── Happy/         # 快乐、兴奋情绪的音乐
├── Pride/         # 成就感、自信情绪的音乐
└── Love/          # 温暖、关爱情绪的音乐
```

## 文件格式要求
- 支持格式：.mp3, .wav, .m4a, .flac
- 时长：建议每首1分钟左右
- 命名：任意名称，程序会自动扫描

## 使用说明
1. 将音乐文件放入对应情绪文件夹
2. 程序会根据用户倾诉内容自动选择合适情绪
3. 从该情绪文件夹随机选择背景音乐
4. 如果本地音乐不可用，会回退到AI生成音乐

## 情绪映射逻辑
- 压力、焦虑 → Quiet (平静)
- 悲伤、低落 → Happy/Love (喜悦/友爱)  
- 愤怒、挫折 → Quiet/Love (平静/友爱)
- 孤独、困扰 → Love/Happy (友爱/喜悦)
- 成功、目标 → Pride/Happy (自豪/喜悦)
