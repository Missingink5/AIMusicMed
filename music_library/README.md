# 音乐库目录结构

**重要说明：本音乐库需要您手动添加音乐文件，系统不会自动生成任何音乐。**

请将您的音乐文件按照以下情绪分类放入对应文件夹：

## 目录说明
```
music_library/
├── Hostility/     # 愤怒、敌对情绪的音乐 (需手动添加)
├── Sad/           # 悲伤、低落情绪的音乐 (需手动添加)
├── Anxiety/       # 紧张、不安情绪的音乐 (需手动添加)
├── Quiet/         # 平和、安静情绪的音乐 (需手动添加)
├── Happy/         # 快乐、兴奋情绪的音乐 (需手动添加)
├── Pride/         # 成就感、自信情绪的音乐 (需手动添加)
└── Love/          # 温暖、关爱情绪的音乐 (需手动添加)
```

## 音乐文件选择建议

### Hostility (敌意情绪)
- **推荐风格**: 舒缓、冷静、平和的音乐
- **目的**: 帮助平复愤怒和敌对情绪
- **示例**: 轻柔的钢琴曲、自然声音、冥想音乐

### Sad (忧郁情绪)  
- **推荐风格**: 治愈、温暖、希望的音乐
- **目的**: 提供情感支持和安慰
- **示例**: 抒情音乐、温柔的弦乐、励志歌曲

### Anxiety (焦虑情绪)
- **推荐风格**: 平静、稳定、减压的音乐
- **目的**: 缓解紧张和不安
- **示例**: 慢节奏音乐、海浪声、深度放松音乐

### Quiet (平静情绪)
- **推荐风格**: 宁静、和谐、冥想的音乐
- **目的**: 维持和加深平静状态
- **示例**: 环境音乐、轻音乐、禅修音乐

### Happy (喜悦情绪)
- **推荐风格**: 轻快、积极、振奋的音乐
- **目的**: 增强正面情绪
- **示例**: 欢快的器乐、正能量歌曲、轻松音乐

### Pride (自豪情绪)
- **推荐风格**: 激励、自信、成就感的音乐
- **目的**: 强化积极的自我认知
- **示例**: 励志音乐、气势恢宏的音乐、成功主题音乐

### Love (友爱情绪)
- **推荐风格**: 温暖、关爱、连接的音乐
- **目的**: 培养爱与连接感
- **示例**: 温暖的民谣、感人的音乐、家庭主题音乐

## 文件格式要求
- **支持格式**: MP3, WAV, M4A, FLAC, OGG
- **推荐时长**: 30秒 - 3分钟
- **音质建议**: 128-320 kbps (MP3)
- **命名规则**: 任意有意义的名称，系统会自动扫描

## 添加音乐文件步骤
1. 准备符合情绪类型的音乐文件
2. 将文件复制到对应的情绪文件夹
3. 运行 `python test_local_music.py` 测试系统识别
4. 系统会自动扫描并统计各目录的音乐文件数量

## 使用说明
1. 系统根据用户倾诉内容自动分析情绪
2. 从对应情绪文件夹随机选择背景音乐
3. 如果对应情绪目录为空，会回退到AI生成音乐
4. 建议每个目录至少添加5-10首音乐以保证多样性

## 情绪映射逻辑
- 压力、焦虑 → Anxiety (焦虑舒缓音乐)
- 悲伤、低落 → Sad (治愈温暖音乐)
- 愤怒、挫折 → Hostility (平复冷静音乐)
- 孤独、困扰 → Love (关爱连接音乐)
- 成功、目标 → Pride (励志成就音乐)
- 开心、兴奋 → Happy (积极快乐音乐)
- 放松、平静 → Quiet (宁静冥想音乐)

---
💡 **提示**: 音乐的选择直接影响冥想体验的质量，建议选择您认为真正有助于情绪调节的音乐作品。
