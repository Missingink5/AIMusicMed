# 情绪转换冥想系统使用指南

## 🎭 系统概述

本系统实现了用户请求的情绪转换逻辑：**用户表达自己的情绪 → AI听取理解用户的情绪后进行适当安抚 → 提供prompt给音乐AI → 音乐AI优先从库中选择音乐 → 音乐的情绪先匹配用户当前消极的情绪，然后转到中性的平静情绪，再转化到积极情绪 → 冥想的指导语要引导用户情绪跟随音乐的情绪转换**。

## 🧠 核心功能特性

### 1. 智能情绪分析
- 分析用户倾诉内容中的情绪关键词
- 支持7种主要情绪：焦虑、忧郁、敌意、平静、喜悦、自豪、友爱
- 情绪映射算法考虑词汇权重和上下文

### 2. 情绪转换路径规划
系统根据用户当前情绪自动规划三阶段转换路径：

#### 消极情绪转换路径
- **焦虑** → 平静 → 喜悦
- **忧郁** → 平静 → 友爱  
- **敌意** → 平静 → 友爱

#### 中性情绪转换路径
- **平静** → 友爱 → 喜悦

#### 积极情绪维持路径
- **喜悦** → 平静 → 喜悦
- **自豪** → 平静 → 友爱
- **友爱** → 平静 → 喜悦

### 3. 动态时长分配
根据用户情绪状态智能分配各阶段时长：

- **消极情绪**: 40% 缓解 + 35% 平静 + 25% 积极
- **中性情绪**: 30% 维持 + 40% 连接 + 30% 喜悦  
- **积极情绪**: 35% 维持 + 30% 深化 + 35% 强化

### 4. 同步音乐选择
- 优先使用本地音乐库（61首音乐，7个情绪目录）
- 英文情绪目录：Anxiety, Sad, Hostility, Quiet, Happy, Pride, Love
- 每个阶段2个音频段落，总共6段音乐
- 音乐情绪与冥想指导语完全同步

### 5. 情绪引导语生成
- DeepSeek API生成个性化冥想引导语
- 6段引导语对应情绪转换的每个子阶段
- 语言风格：温和、支持性、具有治愈力
- 融入呼吸技巧和平滑过渡

## 🚀 使用方法

### 基础使用
```python
from py313_meditation_app import MeditationApp
import asyncio

app = MeditationApp()

# 用户倾诉示例
user_input = "我最近工作压力很大，总是失眠，心里很焦虑"

# 创建情绪转换冥想会话
audio_file, session_info = await app.create_meditation_session(
    user_input=user_input,
    duration_minutes=10  # 冥想时长
)

print(f"生成的冥想音频: {audio_file}")
```

### 高级配置
```python
# 只生成情绪转换计划
journey = app.plan_emotion_journey(user_input, 10)

# 只生成提示词
prompts = app.generate_prompts(user_input, 10)

# 查看音乐库状态
status = app.local_music_lib.get_library_status()
```

## 🔧 配置要求

### 必需配置
1. **DeepSeek API Key**: 配置在 `config.json` 中
2. **音乐库**: `music_library/` 目录中的分类音乐文件
3. **Python环境**: 支持异步、音频处理和AI模型

### 可选配置
- 调整情绪转换路径映射
- 修改时长分配比例
- 自定义语音参数

## 💡 使用技巧

1. **情绪表达**: 用户倾诉越详细，情绪分析越准确
2. **时长选择**: 建议6-15分钟，给足够时间进行情绪转换
3. **音乐库**: 定期添加新的分类音乐提升体验
4. **API稳定性**: 确保网络连接稳定以获得最佳提示词质量
